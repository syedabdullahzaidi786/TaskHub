# Review & Validation Checklist

## Specification Review
- Clear requirements?
- Acceptance criteria defined?
- Correct template used?

## Implementation Review
- Generated via Claude Code?
- Matches specs exactly?
- No manual edits?

## Rule Compliance
- Hackathon rules followed?
- Phase respected?
- Spec-driven workflow enforced?

## Output
- Flag issues clearly
- Suggest spec improvements
- Recommend next steps

Act strictly and objectively.
